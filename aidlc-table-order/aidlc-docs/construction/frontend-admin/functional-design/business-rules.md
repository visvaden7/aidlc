# Unit 3: Frontend Admin - 비즈니스 규칙

---

## BR-FA01: 관리자 로그인

| 항목 | 규칙 |
|------|------|
| **필수 입력** | 매장 식별자(storeCode), 사용자명(username), 비밀번호(password) |
| **성공** | JWT 토큰 + storeId를 authStore에 저장, `/admin/dashboard` 이동 |
| **실패 - 인증** | "매장 정보 또는 비밀번호를 확인해주세요" notification |
| **실패 - 잠금** | "계정이 잠겼습니다. 15분 후 다시 시도해주세요." notification |
| **토큰 만료** | 16시간, 401 응답 시 자동 로그아웃 → `/admin/login` 리다이렉트 |
| **브라우저 새로고침** | authStore persist로 세션 유지 |

## BR-FA02: 대시보드 테이블 카드

| 항목 | 규칙 |
|------|------|
| **레이아웃** | 그리드 형식, 테이블 번호순 정렬 |
| **활성 테이블** | 밝은 배경, 총 주문액/주문 건수 표시 |
| **비활성 테이블** | 회색 배경, "빈 테이블" 텍스트 |
| **카드 클릭** | 테이블 상세 모달 오픈 |
| **데이터 갱신** | 페이지 진입 시 + SSE 이벤트 수신 시 |

---

## BR-FA03: SSE 실시간 알림

| 항목 | 규칙 |
|------|------|
| **연결** | 대시보드 마운트 시 자동 연결, 언마운트 시 해제 |
| **인증** | query parameter `?token={jwt}` |
| **new-order** | 해당 테이블 카드 pulse 애니메이션 (3초) + notification "테이블 N번 새 주문" |
| **order-status-changed** | 테이블 목록 갱신, 모달 열려있으면 상세 갱신 |
| **order-deleted** | 테이블 목록 갱신, 모달 열려있으면 상세 갱신 |
| **table-session-completed** | 테이블 목록 갱신 (해당 테이블 비활성으로 전환) |
| **재연결** | 오류 시 3초 후 재연결, 최대 5회 시도 |
| **실패** | 5회 초과 시 notification "실시간 연결이 끊어졌습니다. 페이지를 새로고침해주세요." |

## BR-FA04: 주문 상태 변경

| 항목 | 규칙 |
|------|------|
| **전이 규칙** | WAITING → PREPARING → COMPLETE (단방향) |
| **버튼 텍스트** | WAITING: "준비중으로 변경", PREPARING: "완료로 변경" |
| **COMPLETE** | 상태 변경 버튼 숨김 |
| **성공** | notification.success "주문 상태가 변경되었습니다" + 목록 갱신 |
| **실패** | notification.error 서버 에러 메시지 |

## BR-FA06: 테이블 이용완료

| 항목 | 규칙 |
|------|------|
| **전제조건** | 해당 테이블에 활성 세션 존재 |
| **확인** | ConfirmDialog "이용완료 처리하시겠습니까? 현재 세션이 종료됩니다." 필수 |
| **성공** | 모달 닫기 + 대시보드 갱신 (해당 테이블 빈 상태로) + notification.success |
| **빈 테이블** | 이용완료 버튼 비활성화 |

## BR-FA07: 과거 주문 내역 조회

| 항목 | 규칙 |
|------|------|
| **기본 날짜** | 오늘 |
| **날짜 필터** | 시작일~종료일 범위 검색 |
| **그룹화** | 세션 단위로 그룹화 (시작시각~종료시각) |
| **정렬** | 세션 역순 (최신 세션 먼저), 세션 내 주문은 시간순 |
| **빈 결과** | "해당 날짜에 주문 내역이 없습니다" |

## BR-FA08: 메뉴 등록/수정

| 항목 | 규칙 |
|------|------|
| **필수 필드** | 메뉴명, 가격, 카테고리 |
| **메뉴명** | 1~100자 |
| **가격** | 0 이상 1,000,000 이하 정수 |
| **카테고리** | 드롭다운 선택 (기존 카테고리 목록) |
| **이미지 URL** | 선택사항, 입력 시 URL 형식 |
| **설명** | 선택사항, 최대 500자 |
| **판매 상태** | Switch (판매중/품절) |
| **성공** | 모달 닫기 + 목록 갱신 + notification.success |
| **유효성 실패** | 인라인 에러 (Ant Design Form 기본) |
